# SalesLoftThis is a repo for managing machine readable artifacts for the [SalesLoft APIs](http://salesloft.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [SalesLoft APIs](http://salesloft.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://salesloft.com)- [Developer](http://salesloft.com)- [Twitter](https://twitter.com/SalesLoft)- [GitHub](https://github.com/SalesLoft)The [APIs.json](https://github.com/api-evangelist/salesloft/blob/master/apis.json) index provides links to all the relevant links for the SalesLoft APIs, including the OpenAPI and Postman if they are present.
